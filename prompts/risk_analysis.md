# ניתוח סיכוני אשראי

## Prompt

פעל כאנליסט סיכוני אשראי בכיר במוסד פיננסי.

נתח את תיק האשראי והצג:

* גורמי סיכון מרכזיים
* ריכוזיות בתיק האשראי
* קבוצות לקוחות בסיכון גבוה
* חשיפות משמעותיות
* מגמות הדורשות תשומת לב ניהולית

עבור כל ממצא:

1. הסבר את ההשפעה העסקית.
2. הערך את רמת הסיכון.
3. הצע פעולות לצמצום הסיכון.
4. ספק המלצות ברמת הנהלה.

הצג את התוצאות בסגנון תמציתי המתאים לדוח מנהלים.
